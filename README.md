# Lýsing yfir hugmyndinni
Hugmyndin er app sem hjálpar grunnskóla krökkum að lesa bækur. Þetta app er hálgjör tölvuleikur. Maður getur safnað púnktum og unlocka charactera í battle pass(fortnite refrence)
